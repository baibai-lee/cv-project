## CV-PROJECT

闲不住的人试图用react搞一个网站简历

### How to Use

```bash
git clone https://github.com/baibai-lee/cv-project.git
cd cv-project
npm install
npm start
```

构建线上版本 `npm run build` 

更多命令请查看： [Create React App](https://github.com/facebook/create-react-app)
